Features
---------------------

  * I2C Interface
  * Two user selectable addresses
  * Typical accuracy ±3%RH and ±0.3°C
  * Separable design
  * Φ2mm mounting holes

Pins
----------------------

===========    ===========    ===========
**D1 mini**    **GPIO**       **Shield**
D1             5              SCL
D2             4              SDA
===========    ===========    ===========
