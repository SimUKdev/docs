Features
---------------------

  * Configurable CS pin, Default: D4 (GPIO0)

Pins
----------------------

===========    ===========    ===========
**D1 mini**    **GPIO**       **Shield**
D5             14             CLK
D6             12             MISO
D7             13             MOSI
D4             0              CS
===========    ===========    ===========
