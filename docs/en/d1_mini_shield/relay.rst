Features
---------------------

  * **NO:** 5A(250VAC/30VDC), 10A(125VAC), MAX:1250VA/150W
  * **NC:** 3A(250VAC/30VDC), MAX:750VA/90W
  * 7 configurable IO, default: D1(GPIO5)

Pins
----------------------

===========    ===========    ===========
**D1 mini**    **GPIO**       **Shield**
D1             5              RELAY
===========    ===========    ===========
