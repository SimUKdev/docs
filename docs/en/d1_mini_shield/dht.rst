Features
---------------------

  * Temperature: -20~60°C (±0.5°C)
  * Humidity: 20-95%RH (±5%RH)

Pins
----------------------

===========    ===========    ===========
**D1 mini**    **GPIO**       **Shield**
D1             5              SCL
D2             4              SDA
===========    ===========    ===========
